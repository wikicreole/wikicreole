= T4 ~WikiParser (T4-WP)

**[[http://web-tec.info/WikiParser/|T4 WikiParser (T4-WP)]]** is a Creole 1.0 parser written in Java.

Features:
* Lightweight and efficient (doesn't use regexp machinery)
* Supports all [[Creole 1.0]] markup with minor deviations and some extensions (see release notes)
* Extensible to allow easy integration into larger frameworks
* Free and open-source

See [[http://web-tec.info/WikiParser/|T4 WikiParser homepage]] for details.

[[http://www.mforum.ru/t4/WikiDemo|Online demo/sandbox]] available for testing.
