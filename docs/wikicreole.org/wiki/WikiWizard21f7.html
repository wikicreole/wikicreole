WikiWizard is an //open source// editor for wikis which combines advanced markup highlighting and wizard dialogs to make wiki editing easier. With wiki creole it would be easy to port WikiWizard to other wiki engines because there are no other inherent dependencies on the underlying wiki engine. 

Learn more about it on [http://www.jspwiki.org/wiki/WikiWizard].

**You** can also [see a screencast demo|http://www.i3g.hs-heilbronn.de/webcasts/wikiwizard/wikiwizard_demo.html]
