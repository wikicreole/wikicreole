This page is reserved for discussion about [implementation] of Creole in TWiki.

The discussion is happening at http://twiki.org/cgi-bin/view/Codev/WikiCreole

The current state of affairs is that WikiCreole support will be built around a TWiki Plugin (http://twiki.org/cgi-bin/view/Plugins/EditSyntaxPlugin). The preliminary version is in the Subversion repository, but has not yet been released to the public.
