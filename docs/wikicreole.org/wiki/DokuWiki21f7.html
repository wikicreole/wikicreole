!!! DokuWiki CREOLE Implementation

[DokuWiki|http://wiki.splitbrain.org/wiki:dokuwiki] supports CREOLE in ''mixed'' mode through a [syntax plugin|http://wiki.splitbrain.org/plugin:creole]. You can choose if CREOLE markup or DokuWiki markup gets precedence for clashing markup in a config setting.

The plugin was written by [Esther Brunner]
