=Sunny Hirai

CEO of MeZine Inc.
www.mezine.com

Creator of a generic markup parser for Ruby.

Originally implemented my own markup because of my dissatisfaction with current wiki markup languages. Later I found out my implementation was about 80% similar to Creole, and the remaining 20% was trivial differences. Creole is a fantastic find and was happy to see that a group came to mostly the same conclusions. Wish I had found it earlier though.

The nice thing with the generic markup parser is that I can implement changes to the language easily. I can add simple rules in seconds.
