Creole 0.4 Core would become what we have now.  An extended section would be added as optional additions.

=== Additions ===

* Tables?
* Indentation?
* Subscript
* Superscript
* Definition Lists
* Underlined?
* Macros?
* Code (inline monospace text)?

Creole is getting mature enough to where some people want to just use Creole on their wikis, and I don't think this should not be possible, just for lack of standard elements.  On the other hand, we don't want to include things in the spec that many wiki engines do not even have.
