== PmWiki Creole

[[http://www.pmwiki.org/|PmWiki]] supports v0.4 Creole in "mixed" mode through an optional component.  A site administrator simply adds an include statement to enable it:

{{{
    include_once('scripts/creole.php');
}}}

There aren't any major conflicts between Creole markup and PmWiki's built-in markup sequences.  PmWiki diverges somewhat in its handling of {{{**}}} bold at the beginning of lines, but it would be possible to update this to strictly follow Creole's specification if that's desired.

More information about PmWiki's Creole module is available at http://www.pmwiki.org/wiki/Cookbook/Creole .
